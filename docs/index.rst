|addon| for the |NCS|
#####################

The |addon| for the |NCS| provides support for developing KNX-IoT applications based on the third-party precompiled KNX-IoT stack.
This stack is included as the KNX-IoT library (version |knx-iot_version|).
In combination with the |NCS|, the |addon| allows for development of KNX-IoT applications.

|addon| for the |NCS|
#####################

The |addon| for the |NCS| provides support for developing KNX IoT applications based on the third-party KNX IoT Point API stack (version |knx-iot_version|).
In combination with the |NCS|, the |addon| allows for development of KNX IoT applications that run over Thread on Nordic Semiconductor devices.

If you are new to KNX or KNX IoT, start with the :ref:`knx_iot_overview` for an introduction to the protocol and its infrastructure.
For architecture and integration details, see :ref:`knx_iot_addon`.
